describe neutron star

  • Neutron star is one of the densest stellar object.
    \(\Rightarrow\) To describe NS, we need General relativity.
     
  • NS is very cold star. (After 1 month, \(T \sim \text{keV}\) while \(\mu \simeq E_F \sim \text{GeV} \))
    \(\Rightarrow\) To describe inner structure of NS, we need Low temperature Nuclear physics.
     
  • But in NS, relative change in the metric over the average distance of baryons is exceedingly small.
    \(\Rightarrow\) To describe interaction in NS, we can use Quantum Field Theory via QCD.

describe neutron star (GR)

Static, spherical symmetric and for convenience, non-rotating metric is

ds ^2 = -e^{2\Phi(r)} dt^2 + e^{2\Lambda(r)} dr^2 + r^2 d\Omega^2

With perfect-fluid matter, we can write two equations.

  1. Energy-momentum conservation (\(\nabla_\mu T^{\mu\nu} = 0\))
  2. Einstein equation (\(G_{\mu\nu} = 8\pi T_{\mu\nu}\))

describe neutron star (GR)

\begin{aligned} \dfrac{dm}{dr} &= 4\pi r^2 \rho \\ \dfrac{dP}{dr} &= - \dfrac{\rho m}{r^2} \left( 1 + \dfrac{P}{\rho}\right) \left(1 + \dfrac{4\pi P r^3}{m}\right) \left(1 - \frac{2m}{r} \right)^{-1} \\ \dfrac{d\Phi}{dr} &= -\dfrac{1}{\rho}\dfrac{dP}{dr}\left(1 + \dfrac{P}{\rho}\right)^{-1} \end{aligned}

Combine all previous equations, we can get coupled equation.

Neutron star cooling

  • Although NS is very cold, temperature is still higher than \(m_\nu\).
    (\(T \sim \text{keV} \gg m_\nu\))
     
  • Thus, neutrinos can escape NS \(\Rightarrow\) Cooling!
     
  • We can observe NS cooling directly. (Heinke & Ho Astrophys.J.719:L167-L171,2010)
    And even real-time! \(\Rightarrow\) Cassiopeia A
