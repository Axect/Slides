Time of MLE

  • Given binary data : \(\mathcal{D} = \{x_1,x_2,\cdots,x_N\}\)
    • \(x_n \in \{0,1\}~(n=1,2,\cdots,N)\)
  • We want to find proper probability distribution of \(\mathcal{D}\)

The simplest way is Maximum Likelihood Estimation

Time of MLE

MLE : Find parameter which maximize likelihood

  • Prior : \(P(\mu)\)
  • Likelihood : \(P(\mathcal{D} | \mu)\)
  • Posterior : \(P(\mu | \mathcal{D})\)

Find \(\mu\) such that \(\displaystyle \frac{\partial}{\partial \mu} P(\mathcal{D}|\mu) = 0\)

Time of MLE

\begin{gathered} \frac{\partial}{\partial \mu} P(\mathcal{D} | \mu) = 0 ~\Rightarrow~\frac{\partial}{\partial \mu} \ln P(\mathcal{D} | \mu) = 0 \\ \vdots \\ \therefore \mu_{ML} = \frac{1}{N} \sum_{n=1}^N x_n = \frac{m}{N} \end{gathered}

For Bernoulli distribution, there is only one parameter \(\mu\)

    distributions::{Distribution, Standard},

fn main() {
    let (h, t) = coin_flip(10);
    println!("{}, {}", h, t);
}

fn coin_flip(n: usize) -> (usize, usize) {
    let mut h = 0usize;
    let mut t = 0usize;
    for _i in 0 .. n {
   		match rand::random() {
        	State::H => { h += 1; },
            State::T => { t += 1; }
        }
    }
    (h, t)
}

    T

            0 => State::H,
            1 => State::T,
            _ => unreachable!()
        }
    }
}

Beta from bayesian

We have background knowledge for some domain.

If data set is given, then we update our knowledge from that.

  • Background knowledge : \(P(\mu)\) (Prior)
  • Data set : \(P(\mathcal{D} | \mu)\) (Likelihood)
  • Bayes theorem:
    $$P(\mu | \mathcal{D}) \propto P(\mathcal{D} | \mu) P(\mu)$$
  • Updated knowledge : \(P(\mu | \mathcal{D})\)

This procedure is called Bayesian inference.

Beta from bayesian

For Bernoulli, we know likelihood.

\(P(\mathcal{D} | \mu) \sim \mu^{\alpha} (1 - \mu)^{\beta}\)

If prior is as follow,

$$P(\mu) \sim \mu^{\alpha'}(1-\mu)^{\beta'}$$

Then posterior also has similar form

$$P(\mu | \mathcal{D}) \sim \mu^{\alpha''}(1-\mu)^{\beta''}$$

Let's start Bayesian inference for Bernoulli distribution

  • Choose hyperparameter : \(a,b\)
  • Make prior : \(P(\mu | a,b) \propto \mu^{a-1}(1-\mu)^{b-1}\)
  • Obtain likelihood : \(\displaystyle P(\mathcal{D} | \mu) \propto \mu^{m} (1- \mu)^{N-m} \equiv \mu^m (1-\mu)^l \)
  • Compute posterior via Bayes theorem
    \(\displaystyle P(\mu | \mathcal{D}) \propto P(m,l | \mu) \cdot P(\mu | a,b)~\Rightarrow~P(\mu|\mathcal{D}) \equiv P(\mu | m,l,a,b)\)
