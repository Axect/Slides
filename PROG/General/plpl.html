Programming Languages

 

Classified by purpose

We can classify programming languages by purpose

  • Web : Javascript, Typescript, Purescript, Elm
  • Game : C#, C++
  • Android : Kotlin, Java, Scala, Javascript
  • Apple : Swift, Javascript
  • AI / Machine Learning : Python, R
  • Data science : R, Python, Scala
  • Server : C/C++, C#, Java, Scala, Kotlin, Go, Rust, Javascript
  • Simulation : C/C++, Fortran

Classified by paradigm

We can classify programming languages by paradigm

  • Procedural (절차적) : C, Fortran, Pascal
  • OOP (객체지향) : C++, Java, C#, Python, Go and etc.
  • Functional (함수형) : Haskell, F#, Lisp, OCaml, Elixir and etc.
  • OOP + FP : Scala, Rust, D, Kotlin, Swift and etc.

Classified by Implementation

  • Native Compiled Language : C/C++, D, Rust, Haskell, Go
  • Use Virtual Machine : Java, Scala, Kotlin, C#, Erlang, Elixir
  • Interpreted Language : Python, Ruby, Javascript

Classified by Feature

  • Easy to programming : Python, Ruby
  • High-Performance (C-level) : Rust, C/C++, Fortran, (Nim, Crystal)
  • High-Performance (Java-level) : Java, C#, D, Go, Haskell, Julia, Swift
  • Parallel : Go, Rust, Kotlin, Swift, Chapel, D, Julia
  • Plot : R, Python, Javascript

(단, \([x]\)는 \(x\) 보다 크지 않은 최대의 정수)

\begin{aligned} a_1 &= 91 \\ a_{n+1} &= \left[\dfrac{a_{n}}{2}\right] \qquad (n\text{은 짝수})\\ a_{n+1} &= a_{n} + 1\qquad (n\text{은 홀수}) \end{aligned}
  1. \(a_{6}\)을 구하시오.
  2. \(a_n\)이 1이 될 때의 \(n\)값을 구하시오.
  3. \(a_n\)이 1이 될 때까지의 합 \(S_n\)을 구하시오.

Channel

// 10_pl/go/11_chan.go
package main

import "fmt"

func main() {
    // Make channel
    c := make(chan int, 100)

    // Send channel
    for i:=0; i<100; i++ {
        s := sum(1000000)
        c <- s
    }
    close(c)

    // Receive channel
    s := 0
    for j := range c {
        s += j
    }
    fmt.Println(s)
}

func sum(n int) int {
    s := 0
    for i:=1; i<=n; i++ {
        s += i
    }
    return s
}
