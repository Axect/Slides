Basic Physics

 

Review: Highschool Physics

역학에서 중요한 3가지 요소는 다음과 같다.

  • \(a\) : 가속도 (Acceleration)
  • \(v\) : 속도 (Velocity)
  • \(s\) : 변위 (Displacement)

 

이들은 다음과 같은 관계를 가진다.

  • \(a = \dfrac{dv}{dt}\) (가속도는 속도의 시간에 따른 변화율)
     
  • \(v = \dfrac{ds}{dt}\) (속도는 변위의 시간에 따른 변화율)

등속도 운동

  • \(a = 0\)
  • \(v = v_0\) (\(v_0\)는 초기속도)
  • \(s = v_0 t\)

등가속도 운동

  • \(a = a\) (\(a\)는 상수)
  • \(v = v_0 + at\)
  • \(s = v_0 t + \dfrac{1}{2}at^2\)

Free particle motion

Q. 찬우가 우주에서 \(20m/s\)의 속력으로 공을 던졌다고 할 때, 3초 후 공의 변위는 얼마인가?
(모든 중력과 마찰은 무시할 수 있다고 가정한다.)

A. \(v = v_0 = 20m/s,~s = v_0 t = 20t~\Rightarrow~s = 60m\)

이것을 코드로 짜는 것은 간단하다.

# 12_physics/01_const.py
def s_gen(v0):
    return lambda t: v0 * t
  
s = s_gen(20)
print(s(3))

Uniformly accelerated motion

Q. 윤수가 20m 높이의 건물에서 공을 놓아 떨어뜨렸다고 할 때, 공은 몇 초 후에 지면에 도달하는가?
(중력가속도는 \(10m/s^2\)이며 모든 공기저항은 무시한다.)

A. \(a = g = 10m/s^2,~v = v_0+gt = 10t,~s=v_0t + \dfrac{1}{2}gt^2 = 5t^2 = 20~\Rightarrow~t = 2\)

이것을 코드로 구현해보자.

# 12_physics/02_unif.py
def s_gen(a, v0):
    return lambda t: v0 * t + 0.5 * a * t**2
  
s = s_gen(10, 0)
t = 0
while s(t) < 20:
  t += 0.1
print(t, s(t))

Differentiation

 

differentiation

Q. 미분계수의 정의는?

A. \(\displaystyle f'(a) =  \lim_{h \rightarrow 0} \frac{f(a+h) - f(a)}{h} \)

Q. 도함수의 정의는?

A. \(\displaystyle f'(x) =  \lim_{h \rightarrow 0} \frac{f(x+h) - f(x)}{h} \)

differentiation

컴퓨터가 미분 계산을 하는 방법은 크게 다음의 2가지가 있다.

  • 수치적 미분 (Numerical Differentiation)
  • 대수적 미분 (Symbolic Differentiation)

 

위의 방법은 빠르고 모든 언어에서 계산 가능하지만 오차가 크고,
아래의 방법은 느리고 특정 언어들에서만 계산이 가능하지만 정확하다.

Numerical differentiation

# 12_physics/03_diff.py
def diff(f, a, h=1e-6):
    return (f(a+h) - f(a)) / h

def f(x):
    return x**2 + 2*x
  
print(diff(f, 1))
print(diff(f, 1, 1e-8))
print(diff(f, 1, 1e-15))

컴퓨터는 극한 계산을 할 수 없으므로 작은 \(h\) 값을 주어 근사한다.

Numerical derivative

# 12_physics/04_deriv.py
import numpy as np
import matplotlib.pyplot as plt

class Derivative:
    def __init__(self, f, h=1e-6):
        self.f = f
        self.h = h

    def __call__(self, x):
        return (self.f(x + self.h) - self.f(x)) / self.h

def f(x):
    return x**2 + 2*x

df = Derivative(f)
x = np.arange(-2, 2, 0.1)

# Write plot code!

Euler method

함수 \(f(x)\)의 도함수를 \(g(x)\)라고 하고, \(h\)가 아주 작다고 하면 다음 식이 성립한다.

g(x) \simeq \dfrac{f(x+h) - f(x)}{h}~\Rightarrow~f(x+h) = f(x) + g(x) \cdot h

즉, 특정 \(x\)에서 \(f(x)\)와 \(g(x)\)의 값을 알고 있다면, \(f(x+h)\)의 값을 구할 수 있다.

이를 Euler method라고 한다.

Euler method

이를 물리에 적용해보면 다음과 같다.

  1. 시간에 대한 가속도를 함수 형태로 알고 있다고 가정한다.
  2. 현재 속도를 이용하여 다음 변위를 예측한다.
    \(s_{t+h} = s_t + v_t \cdot h\)
  3. 현재 가속도를 이용하여 다음 속도를 예측한다.
    \(v_{t+h} = v_t + a(t) \cdot h\)
  4. 시간을 업데이트 한다.
    \(t \rightarrow t+h\)
  5. 1 ~ 4을 반복한다.

Euler method Class

# 12_physics/phys.py
class Phys1D:
    def __init__(self, x, v, f, h=1e-6):
        self.x = x      # Value
        self.v = v      # Value
        self.accel = f  # Function
        self.t = 0      # Value
        self.a = 0      # Value
        self.h = h      # Value

    def calc_accel(self):
        self.a = self.accel(self)

    def update(self):
        self.calc_accel()
        self.x += self.v * self.h
        self.v += self.a * self.h
        self.t += self.h

Revisit DRAG

Q. 윤수가 20m 높이의 건물에서 1kg 공을 떨어뜨렸다고 할 때, 공은 몇 초 후에 지면에 도달하는가?
(중력가속도는 \(10m/s^2\)이며 공기저항력은 \(0.1 v^2\) 이라고 한다. )

# 12_physics/08_drag.py
from phys import Phys1D

def a(p):
    return -10 + 0.1 * p.v**2

ball = Phys1D(20, 0, a, 1e-6)
while ball.x >= 0:
    ball.update()
    print("{:.6f}\t{:.6f}\t{:.6f}".format(ball.x, ball.v, ball.t))
