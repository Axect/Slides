For Loop

# 08_for_print.py
for i in range(1, 11):
    print("i = ", i)
  • i : 변수
  • range(a, b) : a 이상 b 미만
  • for i in range(a, b): 
i를 a부터 b미만까지 바꿔가면서 반복

For Loop (example)

# 09_for_example.py
s = 0
for i in range(1, 6):
    s = s + i
print(s)
  • s = 0
  • i = 1
  • s = 0 + 1
  • i = 2
  • s = (0+1) + 2
  • i = 3
  • s = (0+1+2) + 3
  • i = 4
  • s = (0+1+2+3) + 4
  • i = 5
  • s = (0+1+2+3+4) + 5

# 10_for_example2.py
