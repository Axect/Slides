UP-Down Game

Implement Up-Down Game.

 

  • Computer choose 1 ~ 100 randomly.
     
  • Player guess a number.
     
  • If answer is bigger than guess then print "Up"
    Or lower than guess then print "Down"
    Or exactly same, then finish the game and print "Victory"
     
  • There are five chances. If player can't guess correct answer in 5 times, then finish the game and print "Game over"

high school 1-1:

Find root of Quadratic function
 

Position of Roots

For  \(y=ax^2 + bx + c~(a>0)\),
 

  • \(D \geq 0,~-\dfrac{b}{2a} < p,~f(p) > 0~~\Rightarrow \alpha,\beta < p\)
     
  • \(D \geq 0,~-\dfrac{b}{2a} > p,~f(p) > 0~~\Rightarrow \alpha,\beta > p\)
     
  • \(f(p) < 0~~\Rightarrow \alpha < p < \beta\)

Find root

  1. 1에서 10까지 중에 세 수 \(a,b,c\)를 랜덤하게 뽑는다.
     
  2. 뽑은 \(a,b,c\)로 이차함수 \(y = ax^2 + bx +c\)를 만든다. 단, \(D < 0\) 인 경우에는 다시 \(a,b,c\)를 랜덤하게 뽑도록 한다. 이차함수가 만들어진 후, 프로그램은 축을 알려주어야 한다.
     
  3. 사용자가 \(p\)를 하나 입력한다. 프로그램은 \(f(p)\)를 계산하여 값을 구한다.
     
  4. 앞에서 배운 근의 위치 개념을 이용하여 \(p\)를 움직인다. 일단 처음에는 1단위로 움직여보자. (예를 들어, 만일 두 근이 \(p\)보다 작다는 게 파악되면 \(p\)를 1만큼 빼서 다시 3 ~ 4번을 반복한다.)
     
  5. 만일 \(p\)가 한 번 왔던 곳을 다시 돌아온다면 이번에는 1이 아닌 0.1로 \(p\)를 움직여보자. 그 다음에 또 같은 일이 벌어진다면 0.01, 0.001, ... 이렇게 줄여서 4번을 반복하여 결국 \(f(p)=0\)인 \(p\)를 찾아 출력하여라.

Basic probability
 

Roll a Die

  1. 1에서 6까지 적힌 주사위를 던진다.
     
  2. 10번 던진 후 3이 몇 번 나왔는지 세고, 10으로 나누어 확률을 구한다.
     
  3. 이번엔 100번 던진 후 3이 나온 경우의 수를 세고, 100으로 나누어 확률을 구한다.
     
  4. 이번엔 10000번 던져서 3이 나올 확률을 계산한다.
     
  5. 10000번 던진 결과를 모두 기록하고, 각 횟수까지의 확률을 모두 계산하여 그래프를 그려보자.

 Lotto

  1. 로또 복권을 준비한다. (로또는 1에서 45까지의 수 중 6개의 수를 추측하여 맞추면 된다.)
     
  2. 일단, 랜덤으로 6개의 수를 만들어 로또복권의 답을 구해놓는다. 또한 1개의 보너스 볼을 하나 더 뽑는다. 이 모든 수는 중복되지 않아야 한다.
     
  3. 100,000개의 복권을 긁어 답과 비교하여 등수를 매긴다. 다 맞추면 1등, 5개 맞추고 보너스 볼도 맞추면 2등, 5개 맞추면 3등, 4개 맞추면 4등, 3개 맞추면 5등이다.
     
  4. 각 등수 별로 몇명이 나왔는지 세고, 다음 기준에 따라 당첨금을 계산한다.
    1. 복권 1장은 5000원이다. 따라서 전체 복권 수입액은 500,000,000이다.
    2. 5등 당첨금은 5,000원이고 4등 당첨금은 50,000이다.
    3. 1등 당첨금은 전체에서 4,5등 당첨금을 제외한 금액의 75%이다. 이를 당첨된 수 만큼 분배한다.
    4. 2등 당첨금은 전체에서 4,5등 당첨금을 제외한 금액의 12.5%이다. 위와 동일.
    5. 3등 당첨금은 전체에서 4,5등 당첨금을 제외한 금액의 12.5%이다. 위와 동일.
       
  5. 이를 10번 반복하여 1등부터 3등까지의 평균 당첨금을 계산해보자.

LImit & Fibonacci
 

Fibonacci

피보나치 수열은 다음과 같이 귀납적으로 정의된 수열이다.
\(a_{n+2} = a_{n} + a_{n+1},~a_1 = 1,~a_2 = 1\)

피보나치 수열에는 특이한 성질이 있는데, \(\dfrac{a_{n+1}}{a_n}\)이 점점 어떤 값에 수렴한다는 것이다.

이때 수렴하는 값을 "황금비(Golden Ratio)"라고 부른다.
 

Q. \(n=100\)까지 \(\dfrac{a_{n+1}}{a_n}\)의 값을 그래프로 나타내고 이를 이용해 황금비의 값을 대략적으로 추론하시오.
